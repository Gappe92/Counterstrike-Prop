# Counter-Strike prop

This is the source code for a Counter-Strike prop for playing Airsoft.

## Features
- Hard-Case to protect internals
- 4x4 keyboard, aswell as a switch for input
- audio amplifier and speaker for sound output
- LCD-Display
- uses LIPO battery
- all the time values, codes can be choosen at the start
- code runs on a esp32
- buck converter to get 5V for the esp32

## Tech
- C#
- Networking
